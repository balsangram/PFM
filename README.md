# PFM (Poultry & Food Management Platform)

> A scalable multi-store inventory and order management platform designed for poultry and fresh food businesses. The platform streamlines inventory, order processing, delivery operations, and store management through role-based access and real-time tracking.

---

# 💻 Tech Stack

## Frontend
- React.js
- TypeScript
- Tailwind CSS

## Backend
- Node.js
- Express.js
- MongoDB
- REST APIs

## Additional Services
- JWT Authentication
- Google Maps API
- Firebase Cloud Messaging (FCM)

# 🚀 My Contributions

## 👥 Multi-Role Management

Developed a dynamic Role-Based Access Control (RBAC) system with dedicated dashboards and permissions for different users.

Supported Roles:

- Customer
- Admin
- Store Manager
- Delivery Partner
- Butcher
- Cutter
- Cleaner
- Store Staff

Each role has its own permissions, accessible modules, and operational workflow.

---

## 🏪 Multi-Store Management

Developed a centralized management system supporting multiple stores.

Features include:

- Create and manage multiple stores
- Store-specific inventory
- Store-specific pricing
- Store-wise order management
- Independent staff management
- Store performance monitoring

---

## 📍 Location-Based Store Discovery

Integrated Google Maps to display nearby stores.

Features:

- Detect customer location
- Display stores within a configurable radius
- Show products based on nearest store
- Automatic store selection
- Distance-based product availability
